## Features

- Gulp 4 and nodejs 12.20.0 to 16.0.0
- Bulma 0.9.1
- ES6 support
- Alpine JS

## Usage

1. Install Dev Depedencies

```sh
npm install
```

2. To start development server

```sh
npm run dev
```

3. Build for production

```sh
# build the dist folder
npm run build

# preview the builded files
npx http-server dist
```
